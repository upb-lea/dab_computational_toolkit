dct.sample_requirements.load_dab_specification(dab_configuration_name: str, steps_in_mesh_per_direction: int | None = None) → CircuitDabDTO

Load some predefined DAB specification from different lab samples or papers.

Parameters:

  • dab_configuration_name (str) – configuration name, which is from [“initial”, “everts”, “initial_reversed”]

  • steps_in_mesh_per_direction (int | None) – Steps in each mesh direction (v_1, v_2, p_out), optional, e.g. 2: 2x2x2 = 8 simulations, 4: 4x4x4 = 64 simulations
